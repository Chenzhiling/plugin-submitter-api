## 1. introduction

This project can help you to submit, query, kill flink or spark task in java

## 2. Support version

- Flink 1.13
- Spark 3.1.2

## 3. Support mode

- Flink standalone
- Flink yarn-session
- Flink yarn-per-job
- Spark standalone
- Spark yarn
